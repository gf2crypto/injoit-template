# LaTeX шаблон для журнала INJOIT
Репозиторий содержит LaTeX шаблон статьи для журнала International Journal of Open Information Technologies ([INJOIT](http://injoit.ru/)).
Шаблон основан на шаблоне [IEEEtran LaTeX](http://www.michaelshell.org/tex/ieeetran/).

## Компиляция
**Внимание!** Шаблон не поддерживает компиляцию с помощью `pdflatex`! Используйте `xelatex` (он входит в состав большинства LaTeX дистрибутивов и, скорее всего, уже установлен у вас, если вы используете LaTeX).

Пример компиляции:
```bash
xelatex injoit-rus
biber injoit-rus
xelatex injoit-rus
biber injoit-rus
xelatex injoit-rus
``` 

Рекомендуется для компиляции использовать команду `make` (`make.bat` под Windows). Однако имейте ввиду, что эти команды используют `latexmk`. Возможно, что в `MikTeX` нужно будет её доустановить. 


## Использование шаблона
 * Параметры задаются в файле `config.tex`.
 * Помещайте рисунки в папку `images` и они будут автоматически "подхватываться" шаблоном
 * Библиографию помещайте в файл `%_ru.bib`.
 * По требованиям журнала нужно сформировать библиографию на английском языке для этого англоязычные источники остаются без изменений, а русскоязычные переводятся на английский. Английскую версию библиографии помещайте в файл `%_en.bib`. Не забудьте переименовать ключи библиографических записей в английской версии, чтобы не было дубликатов (см. пример в файле `%_en.bib`).
 * Не забудьте в финальном варианте указать выходные данные статьи и номер первой страницы (см. конец файла `config.tex`)
